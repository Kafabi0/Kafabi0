## 🍃 Featured Project

### SiKaFa Portfolio

A modern, animated personal portfolio built with **React**, **Vite**, and **Tailwind CSS** — featuring an aurora background, a 3D lanyard interaction, and a handful of small, deliberate details along the way.

**Why it matters**
Built as a space to practice motion, restraint, and interaction design at the same time — the goal was a portfolio that feels alive without feeling busy.

**Key features**

- Aurora-style animated background rendered in real time
- Interactive 3D lanyard element built with Framer Motion
- Fully responsive, dark-mode-first layout
- Component architecture built for easy content updates
